# .iffiles
> My personal .dotfiles

## Keyborad Shortcuts via AutoHotkey Scripts
- `Win + Shift + Q`
  - Remap for `Alt + F4` via the [quit.ahk](https://github.com/ifcodesdotnet/.iffiles/blob/master/autohotkey/quit.ahk) script.
- `Win + Shift + C`
  - Shortcut for opening up my .dotfiles in vscode via the [vscode.ahk](https://github.com/ifcodesdotnet/.iffiles/blob/master/autohotkey/vscode.ahk) script.
- `Win + Shift + K`
  - Shortcut for converting text in my clipboard to kebab-case via the [kebab-case-clipboard.ahk](https://github.com/ifcodesdotnet/.iffiles/blob/master/autohotkey/kebab-case-clipboard.ahk) script.
